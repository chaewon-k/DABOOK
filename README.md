## 소독가들을 위한 다독 장려 프로젝트

![Project](/uploads/3f0820e6940864c1e6acdb894bf2d761/Project.jpg)


## 프로젝트 개요

- 진행기간 : 2021.1.11 ~ (ing)
- 만든이 : 소독가들(강세준, 강채원, 김혜민, 장주빈, 최나현, 최낙훈)
- 기획 배경 : HTML, CSS도 모르는 일반인들도 전자책을 만들수 있게 해보자.
- 목표 : GUI와 단축키로 편하게 사용가능한 Epub Editor만들기.
